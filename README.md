# Intro

The project consists of a web platform that allows show all information of the company.

## How to use

To start using this project is very simple, you just have to clone and install the dependencies with NPM. Once this is done, you must copy file ".env.example" in ".env" file and configure the information. The commands to use are the following:

```bash
# Clone this repository
$ git clone https://github.com/cceo-developer/cceo-web-page

# Go into the repository
$ cd cceo-web-page

# Install dependencies
$ npm install

# Run the app
$ npm dev
```

### Prerequisites

The present project was developed using [Vue.js](https://vuejs.org/) 3.* for which it is necessary to have a version of NODE 20.0.* or higher, in addition this project is developing using [Vite](https://vitejs.dev/).
